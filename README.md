## What we do

BrightScale Labs is the parent studio behind a portfolio of focused, single-purpose websites. Each brand is a long-term build: a complete library of calculators, planners, or generators in one vertical, designed for clarity over cleverness, and engineered to load instantly anywhere in the world.

We don't chase trends. We pick categories where existing tools are slow, ad-heavy, or shallow — and rebuild them properly. Every site in the portfolio runs on the same modular design system, the same edge infrastructure, and the same standard for accuracy, accessibility, and disclosure.

## How we build

</div>

| Principle | What it means in practice |
|---|---|
| **Fast** | Static, pre-rendered HTML served from a global edge network. No spinners, no loading states, no waiting. |
| **Reliable** | Cached at the edge, deployed automatically from main. 99.97% rolling uptime across the portfolio. |
| **Clear** | One question per tool. Formula visible. Sources cited. No dark patterns, no upsells, no engagement loops. |
| **Honest about limits** | Calculators illustrate; they don't advise. Every tool carries its disclaimer and points to qualified professionals for decisions that need them. |
| **Built to last** | Modular component library shared across brands. Adding a new tool means adding one JSON file, not redesigning a page. |
| **Privacy first** | Calculators run client-side. Your inputs never leave the browser unless you explicitly submit a contact form. |

The full portfolio shares a math engine, design system, and content schema — so quality, performance, and compliance scale linearly with tool count.

## What we don't do

</div>

We don't take user financial data. We don't run paywalls. We don't sell email lists. We don't auto-play video. We don't gate calculators behind sign-ups. We don't chase pageviews with thin content. We don't claim to give financial, medical, legal, or structural-engineering advice.

The tools illustrate. The user decides.
